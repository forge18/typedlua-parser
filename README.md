# typedlua-parser

Lexer, parser, and AST for TypedLua - a typed superset of Lua.

## Features

- Lexical analysis of TypedLua source code
- Parser for constructing abstract syntax trees
- Support for TypedLua extensions (classes, interfaces, etc.)
- Serde serialization support for AST nodes

## Usage

Add this to your `Cargo.toml`:

```toml
[dependencies]
typedlua-parser = "0.1.0"
```
